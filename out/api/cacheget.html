Description

Retrieve the item from the cache with the specified key.

Parameters

key

Type: string

Description: The key of the item to retrieve.

options

Type: object

Description: Configuration options for this method call. onExpire: Function is the only option available for this method.

Usage

someCache.get('key'); // 'value'

someCache.get('ageLimit'); // 55

someCache.get('things'); // { stuff: 'lots of stuff' }

someCache.get('isIt'); // true
